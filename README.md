# Cloud Atlas — Beta 0.3

Cloud Atlas is a living digital archive and curatorial platform mapping the
invisible infrastructures of the digital cloud.

## Static website files

This folder contains the Beta 0.3 production build. Keep the structure unchanged:

```text
index.html
assets/
README.md
_redirects
```

Do not move the JavaScript or CSS files out of `assets/`.

## Publish with Netlify

Upload the entire folder to Netlify Drop. The `_redirects` file enables direct
links and page refreshes for the React application.
